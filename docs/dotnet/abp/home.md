# ABP是什么

ABP是用于创建现代Web应用程序的完整架构和强大的基础设施! 遵循最佳实践和约定，为你提供SOLID开发经验.

[ABP官网](https://abp.io/)
并且ABP在.Net中的生态已经非常的完善，基本上你遇到的问题都存在解决方案，而且他是开源的你可以参与贡献它

## 为什么推荐学习ABP？

因为您学习了它你可以发现很多很多功能它已经写好了您可以直接拿来就直接使用，它的生态也基本上为它贡献了很多解决方案，它可以使你项目开发效率急速提升，并且安装ABP的DDD领域驱动方案你的项目也可以非常的容易维护，让开发者快速了解并且加入开发。
